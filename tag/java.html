JAVA 扩展

Published: 2016-04-14 13:04:00
By ytwan

In Program.

tags: JAVA

  • 下载mysql-connector-java-5.0.8.zip
  • 解压得到如下文件 mysql-connector-java-5.0.8-bin.jar
  • 把mysql-connector-java-5.0.8-bin.jar 复制到JDK\jre\lib\ext
  • 本计算机的如下文件夹下 C:\Program Files\Java\jdk1.8.0_31\jre\lib\ext
  • DK\jre\lib\ext ，这个文件夹专门放置扩展的jar文件的地方
