In Program.

tags: JAVA

JAVA-JDBC

JAVA中JDBC采用的是分层的思想，其实这个对在处理不同的对象时，采用同一种的API的对生活中很多事情是很有借鉴意义的。

步骤:

1、首先是配置JDBC驱动

驱动程序

2、注册驱动程序

方法一:Class.forName()动态加载驱动程序的类文件到内存中,并自动将其注册java.lang.Class
        Class.forName("com.mysql.jdbc.Driver")
方法二:DriverManager.registerDriver()
        Driver driver = new com.mysql.jdbc.Driver;
        DriverManager.registerDriver(driver)

3、连接数据库

    需要提供驱动器以及数据库的地址名称 ,用户名密码
    DriverManager.getConnection()重载方法创建连接对象 URL Properties
    重载形式一
    Connection con = DriverManager.getConnection(URL)   
    重载形式二   
    Connection con =  DriverManager.getConnection(URL,pro)

4、操作

数据映射 
Statement smt = con.createStatement();
PreparedStatement stm = con.prepareStatement();
查询,提取,等相关操作
    ResultSet result =  smt.executeUpdate()
                        smt.executeQuery()
                        smt.getResultSet()
        结果集 
        result.getInt()     result.getString()  rs.first()

5、清理环境,关闭资源

    conn.close()

备注:

使用到的类与接口:
Connection
Connection pool
Statement
Properties
SQLException 异常
