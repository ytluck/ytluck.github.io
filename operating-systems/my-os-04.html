说明

Linux操作系统为不同的账户登陆提供两种登陆模式，在这两种模式下可以本地登陆和远程登陆。提供7个运行等级 。
登陆模式分为命令行登陆以及图形界面登陆。命令行中提供6个terminal操作界面tty1-tty6。而在图形界面中提供7个，其中第tty7是开机默认等待登陆的图形环境。

    Telnet登陆：简单易用不安全  
    SSH   登陆：
        服务器端设置，服务端是一个守护进程,一般是sshd进程,在后台运行并响应来自客户端的请求。提供了对远程请求的处理，一般包括公共密钥认证、密钥交换、对称密钥加密和非安全连接。包括安装，启动，配置。    
        客户端包括安装和登陆。在Ubuntu系统下，ssh客户端一般是自带的，如果没有可以执行sudo apt-get install openssh-client进行安装，之后执行ps -e|grep ssh如果有ssh-agent那么就表示ssh客户端安装成功了。在Windows操作系统下，有SSH<secure shell client>，Xshell等客户端。

X-window

linux本身没有图形界面,linux现在的图形界面的实现只是linux下的应用程序  
 实现的  
X Window System 简称X或X11.分为X-Service和X-client,是主从架构，可以跨网络和跨平台。 X Window从逻辑上分为三层：X Server、X Client和X协议。   
X-Service：管理显示硬件，驱动等。管理客户端设备  
X-Client ：负责一些事件的处理 ，其中X-Window Manager主要负责全部X-client的管理。常见的WM有GNOME、KDE、Xfce等。其中Display Manager 提供登陆的环境并加载用户选择的WM。

显示引擎

GNOME，Xfce 是以GTK图形界面库为基础。KDE是以Qt图形界面库为基础的。Qt是一个跨平台的C++图形用户界面库
